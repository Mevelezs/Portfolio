Acerca de mí

Hola, Soy Full Stack Web Developer con formación especializada en Diseño Web. Experiencia trabajando con Nodejs, JavaScript, React, Redux, Express, Postgres. Participé en la realización de un e-commerce donde fui parte activa en la construcción del eje principal en el back-end de la página; el contenido de las rutas. Se uso para este fin la integración de una API externa de productos de tecnología usando Sequelize para guardar los datos en una base de datos hecha con PostgresSQL. También el control de los formularios en general y el responsive del dashboard de administrador.

Como proyecto individula realicé una SPA de paises consumiendo una API externa y creando acividades turisticas para los mismos. Usando como tegnologias React, Redux, Postgres, Express, y HTML y para los estilos CSS puro.

Apasionado por la letras, los números y la tecnología. En provecho de las vicisitudes de la pandemia enpecé en éste maravilloso mundo y aunque tengo mucha conección con la filologia y la ingeniería siento que el desarrollo de soluciones tecnológicas es mi camino.
